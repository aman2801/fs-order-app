üß© Simplified App Flow
----------------------

1. User Registration & Login
----------------------------
- User provides email and password
- Backend validates and stores hashed password (bcrypt)
- Returns JWT (24h expiry) + refresh token
- Refresh token allows getting new JWT without re-login

2. Product Catalog
------------------
- Authenticated GET request to /products
- Returns paginated product list
- Optional filters via query params

3. Order Placement
------------------
- Authenticated POST to /orders
- Stores order in MongoDB with:
  - User reference
  - Timestamp
  - IP address
  - Order status (pending)
